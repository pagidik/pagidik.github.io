Combining Style Transfer and Super Resolution for High-Resolution Image Generation

This project combines style transfer and Super Resolution Generative Adversarial Network (SRGAN) to achieve high-resolution style transfer images. The style transfer network was modified to use two images, and the low-resolution image generated was passed through an existing SRGAN model. The resulting images achieved interesting high-resolution style transfer results.

Intrinsic Images by Entropy Minimization

The project aimed to recreate a paper on shadow removal from a single image using entropy minimization. The proposed method eliminates the need for camera calibration or multiple images taken under different lighting conditions. The project experimented with different chromaticity spaces and showed that the proposed method is effective in removing shadows from single images.
